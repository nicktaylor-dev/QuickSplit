# QuickSplit
Tip &amp; Bill calculator.

-----------------------------------------

**Goal**:

I want to take out the mental math of splitting tips/bills among groups of people.

**MVP** (Minimum Viable Product):

- Input a bill
- Input tip amount
- Input number of payees
- Show split total

**Extra Potential Features**

- History save

**Tech Stack**

- Language: Kotlin
- IDE: Android Studio
